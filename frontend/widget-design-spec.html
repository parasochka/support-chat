Все экраны, состояния и интерактивные элементы виджета чата.

Экран 0 — Лаунчер (свёрнутый виджет)

Плавающая кнопка в правом нижнем углу страницы. На мобиле скрывается, пока открыта полноэкранная панель.

Экран 1 — Выбор темы (Topic Picker)

Первый экран после открытия панели. Нет кнопки «Назад». Нет поля ввода. Сессия создаётся в фоне, пока пользователь читает список тем.

Кнопка «Другое»: всегда видна, фиолетовая рамка + glow + bold. Выглядит как особый элемент, не сливается со списком.

Нет: кнопки «Назад», поля ввода, стрипа с подсказками.

Стрип подсказок (suggestions strip): появляется после каждого ответа ИИ. Содержит ≤ 2 вопроса (синие пузыри) + один из двух финальных элементов: либо «Вопрос решён.» (зелёный пузырь, инициатива игрока), либо «Завершить чат» (зелёная кнопка, инициатива ИИ). Оба сразу — никогда. При отправке следующего сообщения стрип очищается мгновенно.

Экран 3 — Автоматическое переключение темы

Когда ИИ определяет, что вопрос относится к другой теме — переключает автоматически, без участия игрока.

Инфо-нотис + повторный запрос
Поддержка
Здравствуйте! Чем можем помочь?
Когда придут деньги с вывода?
Похоже, ваш вопрос про «Вывод средств» — переключаю тему…
После переключения — ответ
Поддержка
Здравствуйте! Чем можем помочь?
Когда придут деньги с вывода?
Похоже, ваш вопрос про «Вывод средств» — переключаю тему…
Выводы обрабатываются в течение 24 часов. Криптовалюта — до 1 часа, карты — до 3 рабочих дней.
Switch note: информационная плашка (без кнопки), остаётся в истории переписки как запись о переключении темы. Цвет — синий (#eef2ff). Переключение полностью автоматическое, пользователь ничего не нажимает.

Экран 4 — Эскалация (передача оператору)

Когда ИИ не может помочь, исчерпан лимит сообщений или игрок просит человека. Ввод блокируется, разговор заканчивается.

Связаться с поддержкой
После эскалации: поле ввода скрыто навсегда (для этой сессии). Кнопка контакта — это ссылка (открывается в новой вкладке). Карточка остаётся в истории. Начать новый чат можно только через лаунчер / кнопку «Назад» — но они ведут к созданию новой сессии. Стрип подсказок скрыт.

Экран 5 — Завершение чата (resolve)

Два сценария завершения: (A) игрок нажал «Завершить чат» — панель закрывается, сессия помечается resolved; (B) игрок нажал закрывающий пузырь «Вопрос решён.» — ИИ прощается, поле ввода прячется.

A: «Завершить чат» — финальное сообщение
Поддержка
Карты, крипто, Skrill — до 15 мин.
Чат завершён. Спасибо за обращение!
B: Игрок нажал «Вопрос решён.» — прощание ИИ
Поддержка
Карты, крипто, Skrill — до 15 мин.
Вопрос решён.
B: После прощания ИИ — ввод скрыт
Поддержка
Карты, крипто, Skrill — до 15 мин.
Вопрос решён.
Рады помочь! Удачных ставок 🍀
Разница между двумя финалами: «Завершить чат» (зелёная кнопка, ИИ-инициатива) — панель схлопывается, ИИ не прощается отдельным сообщением в диалоге (сразу добавляется «Чат завершён» и панель закрывается). «Вопрос решён.» (зелёный пузырь, инициатива игрока) — ИИ генерирует тёплое прощание, потом поле ввода прячется, панель не закрывается — пользователь видит прощание.

Экран 6 — Ошибки

Два типа ошибок: ошибка при отправке (внутри пузыря ответа) и ошибка при загрузке тем (вместо списка).

Ошибка отправки сообщения
Поддержка

Как вывести деньги?
Что-то пошло не так. Попробуйте ещё раз.
Ошибка загрузки тем
Поддержка
Не удалось начать чат. Попробуйте позже.

Экран 7 — Мобильный вид (полноэкранный sheet)

На экранах ≤600px или ≤480px высоты — панель растягивается на весь экран. Лаунчер скрыт. Скругления убраны. Клавиатура учитывается через VisualViewport API.

Mobile особенности: нет border-radius, font-size input = 16px (блокирует iOS auto-zoom), safe-area inset для notch/home indicator, лаунчер скрыт пока открыта панель, max-width пузырей = 88% (против 80% на десктопе).

Схема переходов между экранами

Полный граф состояний виджета.

ЗАКРЫТ
  │
  ▼ [клик на лаунчер]
ЭКРАН 1: Выбор темы  ◄── [кнопка «Назад» / кнопка «Закрыть» → снова ЭКРАН 1 при следующем открытии]
  │                           └── [кнопка «Закрыть»] → ЗАКРЫТ
  │ [клик на тему]
  ▼
ЭКРАН 2: Чат
  │
  ├── [пользователь пишет сообщение]
  │     ├── ИИ отвечает + suggestions (≤2 вопроса + closing/finish)
  │     ├── ИИ детектировал другую тему → ЭКРАН 3: Авто-переключение → ЭКРАН 2
  │     └── Ошибка API → сообщение об ошибке в пузыре (поле остаётся)
  │
  ├── [тап на suggestion bubble] → отправляется как сообщение → цикл выше
  │
  ├── [тап «Завершить чат» (finish, зелёная кнопка)]
  │     └── → ЭКРАН 5A: финальное сообщение → панель закрывается → ЗАКРЫТ
  │
  ├── [тап «Вопрос решён.» (closing bubble, зелёный пузырь)]
  │     └── → ИИ прощается → ЭКРАН 5B: поле скрыто, чат заморожен
  │
  ├── [ИИ эскалирует / лимит сообщений / ключевые слова]
  │     └── → ЭКРАН 4: карточка эскалации, поле скрыто
  │
  └── [кнопка «Назад»] → ЭКРАН 1: выбор темы (новая сессия)
      [кнопка «Закрыть»] → ЗАКРЫТ (сессия дропается)

ЭКРАН 3: Авто-переключение темы
  Показывается switch-note → typing → ЭКРАН 2 с ответом из нужной темы

ЭКРАН 4: Эскалация
  Поле ввода скрыто навсегда для этой сессии.
  Кнопка «Назад» / «Закрыть» → ЭКРАН 1 с НОВОЙ сессией.

ЭКРАН 5A/5B: Завершён
  Поле ввода скрыто.
  Кнопка «Назад» / «Закрыть» → ЭКРАН 1 с НОВОЙ сессией.
