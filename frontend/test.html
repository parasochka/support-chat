NikaBet Support Chat — Test Page

This page embeds the floating support widget (bottom-right). Click the bubble to start a session, pick a topic, and chat. Every interaction is logged to Postgres.

How to tune
  • Edit the CONFIG block at the top of /widget.js to set API_BASE, RECAPTCHA_SITE_KEY, and the sample USER_CONTEXT.
  • Set backend behaviour via environment variables (see README).
  • Escalation buttons point at CONTACT_FORM_URL.
Try
  • Pick Deposits and ask about a failed payment.
  • Ask in Spanish or Russian — answers follow your language.
  • Type “I want to talk to an operator” to force escalation.
